14. 添付アイコン¶

添付ファイルを設定した際に、添付ファイルを示すアイコンの図柄の設定を行います。あらかじめ用意されている図柄以外に、オリジナルの図柄も設定できます。

14.1. 添付ファイルのアイコンを設定する¶

Note

オリジナルの図柄のサイズは、32×32ピクセルにしてください。

  • メニューの をクリックします。

  • をクリックします。

  • 新規登録 をクリックします。

  • 「ファイル拡張子」にファイルの拡張子を入力し、「アイコンファイル名」にオリジナルのアイコンファイルを設定し、 OK をクリックします。 [1]

  • 確認メッセージが表示されたら、 OK をクリックします。

14.2. 添付ファイルのアイコンの設定を修正する¶

  • メニューの をクリックします。

  • をクリックします。

  • 修正したいアイコンの 修正 をクリックします。

  • 各項目を修正し、 OK をクリックします。

14.3. 添付ファイルのアイコンの設定を削除する¶

  • メニューの をクリックします。

  • をクリックします。

  • 修正したいアイコンの 修正 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

脚注

[1]「MIME形式」は入力しなくても問題ありませんが、判る範囲で入力してください。
