4. 設備¶

設備では、設備の名前や、その設備を分類分けする設備区分を登録します。「営業部だけが予約できる設備」「田中さんと加藤さんだけが閲覧できる設備」など、設備ごとにアクセス権が設定できます。

4.1. 設備区分を設定する¶

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • 新規登録 をクリックします。

  • 各項目を入力し、 OK をクリックします。

Note

設備区分コードには任意の数字を入力します。複数の設備区分を登録する場合、設備区分コードが重複しないようにします。また、複数の設備区分を登録した場合、設備区分は設備区分コードが若い順に表示されます。

4.2. 設備区分を修正する¶

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • 修正したい設備区分の 修正 をクリックします。

  • 各項目を修正し、 OK をクリックします。

4.3. 設備区分を削除する¶

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • 削除したい施設区分の 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

Warning

設備区分を削除すると、設備区分に設定されている設備とその設備に登録されている予約も削除されます。

4.4. 設備を設定する¶

登録された設備区分に対して、設備を設定します。設備ごとに、所属単位、またはユーザー単位で、アクセス権の設定ができます。 ここでは、所属単位でアクセス権を設定する方法を紹介します。

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • [設備] タブをクリックし、設備を設定する設定区分を選択します。

  • 新規登録 をクリックします。

  • 各項目を入力し、 アクセス権 をクリックします。

  • 「所属一覧」からアクセス権を設定したい所属を選択し、 追加→ をクリックします。

  • 「所属アクセス権一覧」から 所属 を選択し、設定したいアクセス権をリストから選択します。

  • 変更 をクリックし、 OK をクリックします。

  • OK をクリックします。

Note

個人へのアクセス権設定も所属と同様の操作で 追加・削除 を行います。

Note

設備コードには任意の数字を入力します。複数の設備を登録する場合、設備コードが重複しないようにします。また、複数の設備を登録した場合、設備は設備コードが若い順に表示されます。

Note

画像添付には設備（教室や備品など）の画像を登録します。 参照 をクリックし画像ファイルを選択し、 追加 をクリックします。

<アクセス権の説明>

一覧参照 一つ一つの設備の予約は閲覧できませんが、一日単位や週間・月間単位の一覧は閲覧できます。（予約はできません）
詳細参照 一つ一つの設備の予約内容まで閲覧できます。（予約はできません）
作成 設備の予約ができます。
権限無し 設備を閲覧・予約できなくなります。

4.5. 設備を修正する¶

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • [設備] タブをクリックし、プルダウンメニューから修正したい設備が登録されている 設備区分 を選択します。

  • 修正したい設備の 修正 をクリックします。

  • 各項目・アクセス権を修正し、 OK をクリックします。

4.6. 設備を削除する¶

  • メニューの |config_image| をクリックします。

  • をクリックします。

  • [設備] タブをクリックし、プルダウンメニューから削除したい設備が登録されている 設備区分 を選択します。

  • 削除したい設備の 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

Note

設備を削除すると、設備に登録されている予約も削除されます。
