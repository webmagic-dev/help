11. 用件¶

用件では、伝言の「用件の雛形」を登録できます。あらかじめ用意されている4パターンの雛形以外に、自由に文章を登録できます。

11.1. 行先伝言板の用件を設定する¶

  • メニューの をクリックします。

  • をクリックします。

  • 新規登録 をクリックします。

  • 各項目を入力し、 OK をクリックします。

Note

用件コードには任意の数字を入力します。複数の用件を登録する場合、用件コードが重複しないようにします。また、複数の用件を登録した場合、用件は用件コードが若い順に表示されます。

11.2. 行先伝言板の用件を修正する¶

  • 修正したい用件内容の 修正 をクリックします。

  • 各項目を修正し、 OK をクリックします。

11.3. 行先伝言板の用件を削除する¶

  • メニューの をクリックします。

  • をクリックします。

  • 削除したい用件内容の 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。
