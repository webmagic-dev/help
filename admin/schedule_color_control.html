15. グラフ表示色¶

グラフ表示色では、スケジュールなどのグラフ表示で選択できるグラフの色を設定しておくためのものです。初期値では6色設定されていますが、追加したり削除したりできます。

15.1. グラフ表示色を設定する¶

  • メニューの をクリックします。

  • をクリックします。

  • あらかじめ用意されている色候補群から、任意の色をクリックします。

  • 「ID（0以外）」に任意の数字を入力し、 登録 をクリックします。 [1]

Note

あらかじめ用意されている色候補群の色以外でも、「COLOR」に直接16進数値を入力すれば色の設定が行えます。

  • 確認メッセージが表示されたら、 OK をクリックします。

15.2. 設定したグラフ表示色を削除する¶

  • メニューの をクリックします。

  • をクリックします。

  • 削除したい色の 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

Warning

初期値で作成されている色を削除することもできます。

脚注

[1]IDは他と重複しないようにします。色はIDが若い順に表示されます。
