7. URL リンク¶

URL リンクでは、メニューの で閲覧できるホームページを登録します。
登録する前に、あらかじめ［カテゴリ編集］を登録しておきます。

7.1. URL カテゴリを設定する¶

  • メニューの をクリックします。

  • をクリックします。

  • 「カテゴリ名」「説明」を任意に入力し、 追加 をクリックします。

  • カテゴリ名が表示されたことを確認し、 戻る をクリックします。

7.2. 登録した URL カテゴリを変更する¶

  • メニューの をクリックします。

  • をクリックします。

  • 変更したいカテゴリ名を選択し、 ↑読込↑ をクリックします。

  • 「カテゴリ名」や「説明」を変更し、 変更 をクリックします。

7.3. カテゴリの表示順序を変更¶

カテゴリの表示順序を変更する場合は、順序を変更するカテゴリ名を選択し、 ▲ または ▼ をクリックします。

7.4. 登録した URL カテゴリを削除する¶

  • メニューの をクリックします。

  • をクリックします。

  • 削除したいカテゴリ名を選択し、 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

    Warning

    カテゴリを削除するとカテゴリに登録されているURLリンクも削除されます。

7.5. URL リンクを設定する¶

  • メニューの をクリックします。

  • をクリックします。

  • [URLリンク編集] タブをクリックします。

  • あらかじめ設定しておいた「カテゴリ」を選択し、「リンク名」「URL」「説明」等を任意に入力して 追加 をクリックします。

7.6. 登録した URL リンクを変更する¶

  • メニューの をクリックします。

  • をクリックします。

  • [URLリンク編集] タブをクリックします。

  • 変更したい URL リンクがあるカテゴリを選択、対象となるリンク名を ▲ ▼ の枠内より選択して ↑読込↑ をクリックします。

  • 「リンク名」や「URL」「説明」を変更し、 変更 をクリックします。

7.7. リンクの表示順序を変更する¶

リンクを変更する画面で、順序を変更するカテゴリを選択し ▲ または ▼ をクリックします。

7.8. 登録した URL リンクを削除する¶

  • メニューの をクリックします。

  • をクリックします。

  • [URLリンク編集] タブをクリックします。

  • 削除したい URL リンクがあるカテゴリを選択し、 削除 をクリックします。

  • 確認メッセージが表示されたら、 OK をクリックします。

7.9. リンクを移動またはコピーする¶

  • メニューの をクリックします。

  • をクリックします。

  • [移動とコピー] のタブをクリックし、移動またはコピーさせたいリンク名を選択します。

  • 「コピー・移動先」を選択し、コピーしたい場合は コピー を，移動させたい場合は 移動 をクリックします。

  • 確認メッセージが表示されたら OK をクリックします。
