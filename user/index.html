1. 設定画面(個人)¶

設定画面にはログインしているユーザー個人の設定を行う「個人用設定」と、管理者のみが設定できる「管理者用設定」があります。 [1]
設定を行うことで WebMagic をより快適に利用できるようになります。

個人用設定では、以下の9種類の設定を行うことができます。

  • 1. 個人情報
    • 1.1. 個人情報を変更する
  • 2. スケジュール
    • 2.1. スケジュールのアクセス権を設定する
    • 2.2. スケジュールのアクセス権を削除する
    • 2.3. 時間設定でスケジュールの表示時間を設定する
    • 2.4. スケジュールの画面表示初期値を設定する
  • 3. グループ
    • 3.1. グループを登録する
    • 3.2. グループを修正する
    • 3.3. グループを削除する
    • 3.4. グループにユーザーを登録する
    • 3.5. グループのユーザーを変更する
  • 4. 設備
    • 4.1. 簡単予約設定
    • 4.2. 時間設定
    • 4.3. 初期値設定
  • 5. 掲示板
    • 5.1. 掲示板を設定する
  • 6. ワークフロー
    • 6.1. 雛形申請経路
    • 6.2. 雛形文書
    • 6.3. 初期値設定
  • 7. URL リンク
    • 7.1. URL カテゴリを設定する
    • 7.2. 登録した URL カテゴリを変更する
    • 7.3. カテゴリの表示順序を変更
    • 7.4. 登録した URL カテゴリを削除する
    • 7.5. URL リンクを設定する
    • 7.6. 登録した URL リンクを変更する
    • 7.7. リンクの表示順序を変更する
    • 7.8. 登録した URL リンクを削除する
    • 7.9. リンクを移動またはコピーする
  • 8. シングルサインオン
    • 8.1. シングルサインオンリンクを設定する
  • 9. その他
    • 9.1. その他の各種表示形式を設定する

脚注

[1]「管理者用設定」は、管理者権限があるユーザーのみ表示されます。
