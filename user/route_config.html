6. ワークフロー¶

ワークフローで電子決裁をする場合の承認経路をあらかじめ登録しておくことができます。経路は複数登録できるので、決裁の内容により使い分けることができます。
また、決裁に使用する文書の雛形も登録できます。ワークフローを起動した際の初期設定画面を変更することができます。

6.1. 申請経路を設定する¶

  • メニューの をクリックします。
  • をクリックします。
  • 新規登録 をクリックします。

  • 「申請経路名」を任意に入力し、「ユーザー一覧」から承認者を選択して 追加→ をクリックします。

  • 「優先順序」を変更し、 OK をクリックします。

6.2. 申請経路を修正する¶

  • 各項目を修正し、 OK をクリックします。

  • 確認メッセージが表示されたら OK をクリックします。
