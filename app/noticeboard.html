5. 掲示板¶

全社的に告知したい情報や、部署ごとに周知しておきたい情報を掲示板に掲載します。
「営業部のメンバーのみが読むことができる」といったアクセス権の設定ができます。また、カテゴリで分類することができます。
掲示板の情報は必ず確認するようにしましょう。 [1]

5.1. 掲示板に情報を登録する¶

新しい情報を掲示板に書き込みましょう。書き込んだ後でも編集することができます。

  • メニューの をクリックします。

  • プルダウンメニューより [掲示を新規作成する] を選択し、 実行 をクリックします。

  • 「カテゴリ」を選択し、各項目を入力して OK をクリックします。

  • 指定したカテゴリに情報が登録されます。

Note

ファイルを添付したい場合、 参照 をクリックしファイル名を指定、 追加 をクリックします。

5.1.1. 掲載期間を設定する¶

期間を設定して掲示することができます。
新規作成画面の一番下 [掲載期間] の「記載期間を指定する」をチェックし、掲載期間を設定します。

5.2. 書き込まれている情報を読む¶

掲示板に書き込まれている情報を確認しましょう。

  • メニューの をクリックします。
  • 読みたい掲示の「タイトル」をクリックします。
  • 情報の内容が表示されます。

5.2.1. 掲示文書の状態¶

登録された掲示文書は、件名の左側にある絵の状態で「既読」「未読」「添付」を確認することができます。

・・・ 既読済み情報
・・・ 未読情報
・・・ 添付ファイル付き情報

5.2.2. 既読者一覧を表示する¶

記事を読んだ人、読んでない人の一覧を表示することができます。

  • 掲示の「タイトル」をクリックします。
  • [既読・未読者一覧] タブをクリックします。

5.3. 書き込まれている情報を検索する¶

たくさんの情報が書き込まれている場合、検索機能を使うと読みたい情報をすぐに探し出せます。「タイトル」「登録者」に含まれる文字列から検索します。

  • メニューの をクリックします。

  • キーワードを入力し、 検索 をクリックします。

  • 該当する掲示のみが表示されます。

5.3.1. 検索する前の状態に戻す¶

検索を実行したあと、 リセット をクリックします。

5.3.2. カテゴリごとに絞り込み表示¶

初期値では「すべて」カテゴリが表示されます。カテゴリが複数ある場合、カテゴリを切り替えて確認できます。

  • メニューの をクリックします。

  • ［項目］から、表示させたいカテゴリを選択します。

  • 選択したカテゴリのみが表示されます。

5.4. 登録した情報を修正する¶

掲示板に書き込んだ情報を修正しましょう。

  • メニューの をクリックします。
  • 訂正したい掲示の「タイトル」をクリックし、詳細情報を表示します。
  • 修正 をクリックします。
  • 内容を修正し、 OK をクリックします。

5.4.1. 一括で未読･既読に設定¶

掲示文書を一括で未読・既読に設定できます。

  • 設定したい掲示にチェックをつけて、プルダウンメニューから [チェックされた掲示を既読にする]・[チェックされた掲示を未読にする] を選択します。
  • 最後に 実行 をクリックします。

5.5. 登録した情報を削除する¶

掲示板に書き込んだ情報を削除しましょう。

  • メニューの をクリックします。

  • 削除したい掲示にチェックをつけて、プルダウンメニューから [チェックされた掲示を削除する] を選択し 実行 をクリックします。

  • 削除確認画面が表示されますので、表示されている掲示一覧を確認します。

  • 問題なければ 削除 をクリックします。

脚注

[1]掲示板のカテゴリ作成やアクセス権については 掲示板 をご確認ください。
[2]添付ファイルがある場合、添付ファイル名をクリックすると内容が表示されます。
